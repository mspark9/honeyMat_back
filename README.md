# HoneyMat Backend

AI 기반 식단 관리 서비스 HoneyMat의 백엔드 서버입니다.

## What I built
- RESTful API server with Node.js and Express
- PostgreSQL data modeling and API implementation
- JWT-based authentication
- Food image upload and AI analysis flow
- AWS deployment and image storage setup

## Main Features
- User auth
- Meal CRUD
- Image upload
- AI food analysis
- Nutrition data storage
- Daily summary update

## Tech Stack
- Node.js
- Express
- PostgreSQL
- JWT
- AWS
- OpenAI API

## Architecture
Client -> API Server -> PostgreSQL / OpenAI / AWS
